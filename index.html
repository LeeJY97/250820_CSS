배우고 있는 기술 스택

Java & Spring

객체지향 개념을 바탕으로 백엔드 시스템을 개발하는 방법을 배우고 있습니다. Spring 프레임워크를 활용하여 안정적인 서버를 구축하는 것을 목표로 합니다.

AWS & Docker

클라우드 환경에 대한 이해를 높이고 있습니다. Docker를 사용한 컨테이너 기반 개발 환경 구축과 AWS 서비스(EC2, S3 등)를 활용한 인프라 관리 능력을 키우고 있습니다.

SQL & Database

데이터베이스의 중요성을 인식하고, SQL을 통해 데이터를 효율적으로 다루는 방법을 익히고 있습니다. 데이터 관리 역량을 향상시킬 계획입니다.
